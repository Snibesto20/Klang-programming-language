# Klang - a new interpreted language dominating every other in fast pace!

**General information:**
<br>
Klang is a **high level**, **dynamically typed**, **built on Python**, **interpreted** language that has a syntax combining Python and C++ inhereting strengths from both languages resulting in being the best!


**Features:**
<br>
• A hybrid syntax between Python and C++ resulting in readability, but being brace based.
<br>
• Input and output *(terminal)*.
<br>
• Branching algorithms *(If, else)* with logical operators (|, &&, !)
<br>
• For and while loops *(C++ style)*!
<br>
• Variable declaration.
<br>
• Primitive data types *(string, integer, booleans, floats)*.
<br>
• Data structures *(arrays, stacks)*.
<br>
• Built in sorting algorithm methods for arrays *(insertion sort, bubble sort...)*.
<br>
• Syntax sugar 🥳! Printing of containers, *x *= 2* instead of *x = x * 2*, string interpolation *print("Hello {name}!")* instead of print("Hello ", name, "!").
<br>
• Automatic type inference for inputs.
